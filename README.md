# CI/CD Pipeline using Jenkins, Docker & AWS EC2

## Overview
This project automates application deployment using Jenkins CI/CD pipeline.  
Whenever code is pushed to GitHub, Jenkins builds a Docker image and deploys the application on an AWS EC2 instance.

## Tech Stack
- Jenkins  
- Docker  
- AWS EC2  
- GitHub  
- Python (Flask)  

## Workflow
1. Push code to GitHub  
2. Webhook triggers Jenkins  
3. Jenkins pulls latest code  
4. Docker image is built  
5. Container is deployed on EC2  
6. Application goes live  

## Key Features
- Automated deployment (no manual steps)  
- Docker-based consistent environment  
- Fast and reliable releases  

## Result
Reduced deployment time by ~70% and eliminated manual errors.
